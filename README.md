# **docker-app-backend**

## Aplicação backend feita utilizando a biblioteca NestJs, com persistência de dados usando o PostgreSQL.

Para rodar a aplicação siga os passos a seguir:

### PowerShell
    docker compose build; docker compose up -d

### CMD
    docker compose build && docker compose up -d
